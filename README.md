## 如何使用？

将此项目克隆下来安装依赖后在service目录运行你想要的代码

```shell
pip install -e .
```

开机自启动？制作快捷方式放到启动目录里

记得回来看看是否有更新

## 小技巧

- 文件管理器地址栏输入 `启动` 可快速跳转到启动目录

- 按住alt并拖放可快速创建快捷方式
